# SEO Strategy --- Plombier Lyon

Site cible : https://lyon.plomberie-roche.fr/

Objectif : Créer une **stratégie SEO complète pour dominer les
recherches Google liées à la plomberie à Lyon** et générer un maximum
d'appels.

------------------------------------------------------------------------

# 1. Objectifs SEO

Objectifs principaux :

-   apparaître en **top 3 Google** sur les recherches locales
-   générer **appels entrants**
-   capter le trafic des recherches **urgence plomberie**

Exemples de requêtes stratégiques :

-   plombier lyon
-   plombier urgence lyon
-   plombier dépannage lyon
-   fuite eau lyon
-   débouchage canalisation lyon

# 7. Optimisation Google Business Profile

Indispensable pour le SEO local.

Actions :

-   obtenir 100+ avis clients
-   ajouter photos réelles
-   répondre aux avis
-   publier régulièrement

Description recommandée :

Plombier à Lyon spécialisé dans le dépannage urgent : fuite d'eau,
débouchage canalisation, réparation chauffe-eau. Intervention rapide
24h/24 dans toute la métropole de Lyon.

------------------------------------------------------------------------

# 8. SEO technique

Optimisations techniques nécessaires :

-   site rapide (chargement \<2 secondes)

-   sitemap XML
-   balises title optimisées
-   balises H1/H2 structurées
